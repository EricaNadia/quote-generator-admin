# Quote Generator Admin

AI-powered quote generation and publishing admin tool.

## Live App
https://quote-generator-admin-e9192186.base44.app

## What It Does
- Generates 4 AI quotes on any topic
- Save, discard, or publish each quote individually
- Publishing sends the quote to the Wix display site via a secure backend function

## Tech Stack
- React (frontend, hosted on Base44)
- Base44 backend function (publishQuote — secure Wix integration layer)
- Base44 AI integrations (text generation + image generation)
